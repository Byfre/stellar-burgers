# Stellar Burgers

Одностраничное приложение (SPA) для сервиса заказа бургеров, созданное с использованием библиотеки React и Redux Toolkit для управления состоянием. Реализует полный цикл авторизации, оформление заказа, конструктор бургера с перетаскиванием ингредиентов, ленту заказов в реальном времени и личный кабинет.

Проект покрыт юнит-тестами (Jest) и интеграционными тестами (Cypress).

## Тестирование

### Интеграционные тесты (Cypress)

- **Страница конструктора бургера:**
  - Настроен перехват запроса на эндпоинт `api/ingredients` с моковыми данными
  - Протестировано добавление ингредиентов в конструктор (булок и начинок)
  - Проверена работа модальных окон: открытие, закрытие по крестику и по клику на оверлей
  - Протестировано создание заказа: моковые данные пользователя и заказа, авторизация, проверка номера заказа в модальном окне, очистка конструктора после успешного заказа

### Юнит-тесты (Jest)

- Проверена правильная инициализация `rootReducer`
- Протестированы редьюсеры слайса `burgerConstructor`:
  - Добавление ингредиента
  - Удаление ингредиента
  - Изменение порядка ингредиентов в начинке
- Протестированы редьюсеры слайса `ingredients`:
  - Обработка экшена `Request` — установка `isLoading` в `true`
  - Обработка экшена `Success` — запись данных в стор и установка `isLoading` в `false`
  - Обработка экшена `Failed` — запись ошибки в стор и установка `isLoading` в `false`.
